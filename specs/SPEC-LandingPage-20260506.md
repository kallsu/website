# Feature: Landing Page for Call-To-Action

## Requirements

 - Create a new page called `courses.html`
 - The new page is placed on the root level of the website
 - Reuse and extend the existing CSS stylesheet
 - Mock the images
 - Create the Call-To-Action (CTA) button as "Register"
 - Mock the link for the CTA button, will be added manually later
 - Use a simple and clear layout to display the courses
 - For each course must be clear that exists a FREE and PAID version

## Business rules

 - Course:
   - Build an AI app in 5 steps (and why step 6 breaks everything)
   - Agentic AI: hype vs reality for developers
   - .NET Application Development 101

 - All the courses has a FREE and PAID version.
 - The FREE version is 60-90 mins online (default), or on sponsor venues (optional).
 - The PAID version is only face-2-face, 3-4 hours.
 - The FREE version courses are used as lead-generation hooks
 - Trainer value: “The bridge between real-world architecture and teachable skills.”
 - Differentiation Position: the trainer is positioning as “Architecture mentor”, and “Industry practitioner” and “AI + System design expert”.

## Acceptance criteria

 - All the courses must be present
 - All the courses must have their title
 - CTA button must be present

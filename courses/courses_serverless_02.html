Serverless 101 - Beginner Level

What you will learn

  • You will learn WHAT IS the main concepts of Serverless, Event-Driven and States
  • You will learn WHY Serverless is now commonly used
  • You will learn WHAT and WHY about the usage of specific service by AWS and Azure real-cases as example
  • You will learn HOW-TO use and WHEN use them using the AWS and Azure real-cases
  • You will learn WHAT are the Well-Architected framework and their pillars
  • You will learn WHY you should be "Well-Architected"

Target Audience

This course is especially designed for Developers, Freelancer, Technical Team Leader, Solution Architect and / or people which are directly involved in the implementation of a serverless architecture.

Duration

4 hours

Course Structure

Part 1

  • Serverless introduction
  • Event-Driven concepts
  • State-less/full concepts
Part 2
  • Azure Serverless Architecture and Services Test-Case #1
  • Azure Serverless Architecture and Services Test-Case #2
Part 3
  • AWS Serverless Architecture and Services Test-Case #1
  • AWS Serverless Architecture and Services Test-Case #2
Part 4
  • Azure Well-Architected
  • AWS Well-Architected
